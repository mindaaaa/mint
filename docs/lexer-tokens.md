# MINT 렉서 토큰

## 키워드

- `plant`: 변수 선언
- `breeze`: 조건문
- `bloom`: 반복문
- `petal`: 함수 선언
- `gift`: 반환
- `sparkle`: 출력
- `softly`: 문 사이를 잇는 연결어

## 리터럴

- 문자열: 큰따옴표(`"`)로 감싼 텍스트 (예: `"gentle"`)
- 숫자: 정수 및 소수 (예: `0`, `42`, `3.14`)

## 연산자 및 구두점

- 수학 연산: `+`, `-`, `*`, `/`
- 비교 연산: `==`, `!=`, `<`, `<=`, `>`, `>=`
- 대입: `=`
- 그룹핑: `(`, `)`, `{`, `}`
- 기타: `,`, `.`, `;`

## 위치 정보

모든 토큰에는 `line`, `column` 정보가 포함되어 에러 메시지에서 정확한 위치를 가리킬 수 있습니다.
